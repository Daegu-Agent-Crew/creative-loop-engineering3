# CLE3 평가 루브릭 v2 — 닥터슬럼프풍

## 공통 규칙
- 각 Phase: 5항목 × 10점 = **50점 만점**
- 통과 기준: **≥40점** (Phase 5는 **≥42점**)
- 미달 시 자동 롤백 및 이전 Phase 재실행

---

## Phase 1 — Story (스토리 각색, 4단계)

### [A] 원작 분석
| 항목 | 설명 |
|------|------|
| 사건 추출 | 원작에서 에피소드 핵심 사건을 누락 없이 추출했는가 |
| 로그라인 | 아이러니, 머릿속 그림, 타겟 독자를 포함한 1~2문장 요약인가 |
| 캐릭터 아크 | 각 인물의 감정 변화 곡선이 명확한가 |
| 가설 배당 | PRD 가설이 구체적이고 측정 가능한 변수로 정의되었는가 |

### [B] 구조 설계
| 항목 | 설명 |
|------|------|
| 3막 구조 | Act 비율(1:2:1 근사)이 적절한가 |
| 비트 시트 | Save the Cat 비트가 페이지에 매핑되었는가 |
| 감정 곡선 | 고저가 있어 단조롭지 않은가 |
| 페이지 배당 | 풀페이지 임팩트 위치가 의도적인가 |

### [C] 장면 작성
| 항목 | 설명 |
|------|------|
| 장면 목적 | 각 장면이 에피소드에서 하는 역할이 명확한가 |
| 대사 품질 | 캐릭터 성격에 맞는 자연스러운 한국어 대사인가 |
| 감정 설계 | 장면 내 감정 변화가 설계되었는가 |
| 연출 메모 | 색감/분위기/특수 연출이 구체적인가 |

### [D] 패널 분해
| 항목 | 설명 |
|------|------|
| 감정/연출 메모 | Phase 4 이미지 생성에 직접 사용 가능한 수준인가 (★ 핵심) |
| 카메라 다양성 | 앵글이 다양하고 의도적인가 |
| 새 독자 친화 | 첫 3페이지가 이전 화 없이 이해 가능한가 |
| 클리프행거 | 마지막 페이지 훅이 효과적인가 |

## Phase 2 — Characters (캐릭터 디자인)
| 항목 | 설명 |
|------|------|
| 외모 일관성 | 캐릭터 외모 설명이 에피소드 내 일관적인가 |
| 토리야마 스타일 | 닥터슬럼프풍 특성(둥근 윤곽, 셀 채색, 과장된 표정)이 반영되었는가 |
| 시각 구분성 | 캐릭터 간 실루엣만 봐도 구분되는가 |
| 색감 설계 | 캐릭터별 코어 색상이 명확하고 조화로운가 |
| style_notes 품질 | Phase 4 이미지 생성에 직접 사용 가능한 수준인가 |

## Phase 3 — Storyboard (스토리보드)
| 항목 | 설명 |
|------|------|
| 장면 구성 | 각 페이지의 패널 구성이 효과적인가 |
| 카메라 다양성 | 앵글이 다양하고 의도적인가 (토리야마풍 동적 구도) |
| 페이지 흐름 | 시각적 내러티브 흐름이 자연스러운가 |
| 연출 효과 | 풀페이지 임팩트와 여백의 대비가 효과적인가 |
| 대본 반영 | 감정/연출 메모가 스토리보드에 정확히 반영되었는가 |

## Phase 4 — Panels (패널 이미지 생성)
| 항목 | 설명 |
|------|------|
| 스타일 일관성 | 모든 패널이 닥터슬럼프풍으로 통일되어 있는가 |
| 캐릭터 일관성 | 캐릭터가 시트와 동일하게 유지되는가 |
| 감정 연출 | 대본의 감정 메모가 시각적으로 표현되었는가 |
| 한글 렌더링 | 이미지 내 한글 텍스트가 정확히 렌더링되었는가 |
| 구도 정확도 | 스토리보드의 카메라 앵글·구도가 정확히 구현되었는가 |

## Phase 5 — QA (품질 보증 및 완성)
| 항목 | 설명 |
|------|------|
| 전체 일관성 | 에피소드 전체 톤, 화풍, 내러티브가 일관적인가 |
| 이미지 품질 | 해상도, 색감, 디테일, 선 품질이 충분한가 |
| 스토리 정합성 | 최종 결과물이 스토리와 정합하는가 |
| 한글 오류 | 오탈자, 맞춤법, 폰트 오류가 없는가 |
| 완성도 | 배포 가능한 수준의 최종 완성도인가 |
