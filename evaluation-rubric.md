# CLE3 평가 루브릭

## 공통 규칙
- 각 Phase: 5항목 × 10점 = **50점 만점**
- 통과 기준: **≥40점** (Phase 5는 **≥42점**)
- 미달 시 자동 롤백 및 이전 Phase 재실행

---

## Phase 1 — Story (스토리 각색)
| 항목 | 설명 |
|------|------|
| 원작 충실도 | 원작의 핵심 장면과 메시지를 정확히 반영하는가 |
| 각색 자연스러움 | 웹툰 포맷에 맞게 자연스럽게 각색되었는가 |
| 대사 품질 | 캐릭터 성격에 맞는 자연스러운 대사인가 |
| 장면 전개 | 장면 간 흐름과 템포가 적절한가 |
| 캐릭터 일관성 | 원작의 캐릭터 성격이 유지되는가 |

## Phase 2 — Characters (캐릭터 디자인)
| 항목 | 설명 |
|------|------|
| 외모 일관성 | 에피소드 내 캐릭터 외모 설명이 일관적인가 |
| 성격 표현 | 시각적으로 성격이 드러나는 디자인인가 |
| 원작 부합 | 원작 묘사에 부합하는 캐릭터 디자인인가 |
| 시각 구분성 | 캐릭터 간 충분한 시적 구분이 있는가 |
| 스타일 준수 | 통일된 아트 스타일 가이드를 따르는가 |

## Phase 3 — Storyboard (스토리보드)
| 항목 | 설명 |
|------|------|
| 장면 구성 | 각 페이지의 장면 구성이 효과적인가 |
| 페이지 흐름 | 페이지 간 시각적 내러티브 흐름이 자연스러운가 |
| 연출 효과 | 칸 크기, 앵글, 레이아웃으로 연출 효과를 내는가 |
| 원작 충실도 | 스토리보드가 원작 장면을 충실히 반영하는가 |
| 화풍 적합성 | 스토리보드가 최종 화풍과 조화로운가 |

## Phase 4 — Panels (패널 이미지 생성)
| 항목 | 설명 |
|------|------|
| 프롬프트 정확도 | 생성 프롬프트가 스토리보드를 정확히 반영하는가 |
| 화풍 일관성 | 전체 패널의 화풍이 일관적인가 |
| 캐릭터 일관성 | 캐릭터가 에피소드 내내 일관되게 그려졌는가 |
| 한글 렌더링 | 이미지 내 한글 텍스트가 정확히 렌더링되었는가 |
| 장면 연출 | 감정, 분위기, 액션이 효과적으로 표현되었는가 |

## Phase 5 — QA (품질 보증 및 완성)
| 항목 | 설명 |
|------|------|
| 전체 일관성 | 에피소드 전체의 톤, 화풍, 내러티브가 일관적인가 |
| 이미지 품질 | 해상도, 색감, 디테일이 충분한가 |
| 스토리 정합성 | 최종 결과물이 스토리와 정합하는가 |
| 한글 오류 | 오탈자, 맞춤법, 폰트 오류가 없는가 |
| 완성도 | 배포 가능한 수준의 최종 완성도인가 |
