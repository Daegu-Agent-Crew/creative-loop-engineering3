# EP003 구조 설계 (v3)

## 총 페이지: 15p

## 콜드 오픈 전략

- **방식:** (b) 부조화 삽입
- **첫 컷:** 왕먀오 1인칭 시점. 갈색 일색의 황량한 평원이 펼쳐져 있다. 하늘에는 태양이 **없다**. 그런데 왕먀오의 안경 렌즈 가장자리에 카운트다운 숫자가 잠시 비친다 — 게임 세계와 현실의 카운트다운이 연결되어 있음을 암시.
- **훅:** "이게 게임이라고? 하늘에 태양이 없는데 어떻게 게임이 돌아가지?" — 미스터리+부조화 동시 제공
- EP002의 끝(초대장 발견)에서 바로 이어짐

## McKee Gap 맵

| 장면 | 캐릭터 기대 | 실제 결과 | Gap(상황 변화) |
|------|-----------|----------|---------------|
| 게임 진입 (p1) | 왕먀오: 일반적인 VR 게임 환경 | 태양 없는 갈색 황무지. 물리 법칙이 다름 | 기대 → 완전한 부조화 |
| 주희 조우 (p2~3) | 왕먀오: NPC가 퀘스트를 줄 것 | 주희는 퀘스트가 아닌 철학적 질문을 던짐 | 게임 → 실존적 대면 |
| 주희와 대화 (p3~4) | 왕먀오: 과학으로 설명 가능 | 주희: "네 안경은 네 믿음만 보여줄 뿐" | 자신감 → 불확실성 |
| 안정기 체험 (p5~7) | 독자: 위험한 세계 | 갑자기 아름다운 안정기 — 두 태양의 장관 | 긴장 → 안도 (가짜 승리) |
| 관측 결과 (p8) | 상구: 정상 궤도 확인 | 궤도가 완전히 어긋남 | 안심 → 공포 |
| 세 번째 태양 (p9) | 왕먀오: 계산이 맞을 것 | 세 번째 태양 출현 — 계산이 무의미 | 과학 → 무력함 |
| 계산 시도 (p10) | 왕먀오: 수학으로 풀 수 있다 | 수식이 발산. 풀리지 않음 | 오만 → 굴복 |
| 세 태양 상승 (p11~13) | 주희: 이번엔 견딜 수 있다 | 세 태양이 동시에. 문명 소멸 | 통치 → 멸망 |
| 게임 종료 (p14~15) | 왕먀오: "게임일 뿐" | 안경에 카운트다운 재출현 — 현실과 연결됨 | 게임 → 현실의 공포 |

## 풀페이지 배치

- **p1:** 콜드 오픈 — 황량한 평원 + 안경에 카운트다운 (부조화 임팩트)
- **p7:** Midpoint — 안정기 축제, 두 태양의 아름다움 (가짜 승리의 시각적 절정)
- **p10:** 침묵 페이지 — 왕먀오의 계산 실패, 과학의 한계 ★
- **p12:** 세 태양 동시 출현 — 시각적 폭발 (H4 시각 압도 최고점)
- **p15:** 현실 복귀 — 안경에 카운트다운 (클리프행거)

## 침묵 페이지

- **p10:** 왕먀오가 바닥에 무릎 꿇고 앉아 수식을 쓰다 멈춘다. 하늘을 올려다본다. 대사 없음. 흩어진 계산 용지와 하늘의 붉은 기운만. — 과학이 실패한 순간의 공허함을 침묵으로 전달. H4 부 가설의 "여백" 축.

## EP002 연결 설계

- **EP002 끝:** 카운트다운이 거의 0이 되고, 삼체 게임 초대장 발견
- **EP003 시작:** 왕먀오가 초대장을 열고 게임에 접속하는 순간
- **카운트다운 ↔ 게임 연결 암시:**
  - p1: 안경에 카운트다운 잔상 (게임 진입과 함께 사라짐 → 연결 암시)
  - p10: 계산 실패 순간, 카운트다운 숫자가 하늘에 잠시 겹쳐 보임
  - p15: 게임 종료 후 안경에 카운트다운 재출현 → "게임이 끝나도 카운트다운은 계속된다"
