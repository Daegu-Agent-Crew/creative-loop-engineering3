# EP002 구조 설계 (v3)

## 총 페이지: 15p

## 콜드 오픈 전략

- **방식:** (a) 미스터리 + EP001 회수
- **첫 컷:** EP001 p1의 양동 책상을 동일한 구도로. 그러나 이번엔 살아있는 사람(왕먀오)이 그 앞에 서 있다. 죽은 자의 자리에 산 자가 섰다는 대비.
- **훅:** "이 사람은 왜 죽었지? 저 노트의 글씨는... 그리고 모니터에 왜 숫자가?" — 미스터리 + 불안
- EP001의 콜드 오픈과 시각적으로 미러링되면서 "이야기가 계속되고 있다"를 즉각 전달

| p1~2 | **Theme Stated ★** | 왕먀오의 아내 쯔쯔가 전화로 "여보, 늦지 마요. 아이가 아빠 그림 그렸어요" — 가족의 따뜻함. 왕먀오는 "응, 금방 갈게" 하지만 모니터의 잔상 숫자를 보지 못한다. | 3 |
| p2 | Setup | 왕먀오의 연구실. 나노 재료 실험 데이터. 정상적인 과학자의 하루. 안경, 흰 가운, 정돈된 책상. | 3 |
| p3 | **Catalyst** | 카운트다운 첫 출현. 왕먀오의 시계 디스플레이에 **"1199:59:59"**가 1초간 깜빡임. 왕먀오만 본다. | 7 |
| p3~4 | Debate | 왕먀오, 카운트다운을 다시 보려 하지만 사라짐. "내가 미쳤나?" 불안. 동료에게 물어보지만 아무도 못 봄. | 5~6 |
| p4 | **Break into Act 2** | 스치친이 연구실 문을 열고 들어온다. "왕먀오 교수? 경찰이에요. 당신, 위험할 수도 있습니다." | 5 |
| p4~5 | B Story | 스치친과 왕먀오의 첫 만남. 완전히 다른 두 사람 — 형사의 직관 vs 과학자의 이성. 서로의 언어가 다름. | 5~6 |

## McKee Gap 맵

| 장면 | 캐릭터 기대 | 실제 결과 | Gap (상황 변화) |
|------|-----------|----------|---------------|
| 콜드 오픈 (p1) | EP001에서 게임이 시작될 것 | 왕먀오가 [N]을 눌러 게임을 거부 | 기대 → 거부 — 그러나 카운트다운은 시작됨 |
| 일상 (p2) | 왕먀오: 평범한 하루 | 쯔쯔의 전화 + 모니터 잔상 | 안심 → 미묘한 불안 |
| 카운트다운 발견 (p3) | 왕먀오: 내 눈이 잘못됐나 | 다른 화면에서도 출현 | 부정 → 인정 — "이건 진짜다" |
| 스치친 등장 (p4) | 왕먀오: 혼자 처리할 수 있다 | 스치친: "당신은 위험합니다" | 고립 → 강제 연결 |
| 버디 조사 (p5~6) | 스치친: 단순 패턴 | 왕먀오: 연구와 카운트다운의 상관관계 발견 | 표면 → 심층 — 과학자의 분석력이 형사의 직관과 결합 |
| ★ Midpoint (p6) | 왕먀오: 이해하면 해결할 수 있다 | 연구 = 카운트다운 감소 → "연구를 멈춰야 산다" | 이해 → 딜레마 — 지식 vs 생존 |
| 연구 중단 (p7~8) | 왕먀오: 멈추면 끝난다 | 카운트다운 정지 — 가짜 승리 | 공포 → 안도 |
| ★ All Is Lost (p9) | 왕먀오: 멈췄으니 안전하다 | "연구 멈춘 과학자도 죽었어" | 안도 → 절망 — 멈춰도 죽는다 |
| 가족 위협 (p10~11) | 왕먀오: 가족은 모른다 | 쯔쯔가 숫자를 본다 | 비밀 → 폭로 — 더 이상 숨길 수 없다 |
| 결단 (p12~14) | 왕먀오: 도망칠 수 있다 | "게임을 해야만 정보를 얻을 수 있다" | 도망 → 직면 — 유일한 방법 |
| 클리프행거 (p15) | 왕먀오: [Y]를 누르면 답이 있다 | 개인 카운트다운이 부여됨 — "719:59:59" | 결의 → 새로운 공포 — 게임이 곧 생명 |

## 풀페이지 배치

- **p1:** 콜드 오픈 — 양동의 책상 + 왕먀오 (EP001 미러링)
- **p8:** ★ 침묵 페이지 — 정지한 카운트다운, 멈춘 시계
- **p15:** ★ 클리프행거 — "당신의 카운트다운: 719:59:59"

---

## 침묵 페이지

- **p8:** 왕먀오가 연구를 중단한 후, 카운트다운이 멈춘 순간. 방 안의 모든 시계가 멈춰 있다. 대사 없음. 창밖으로 쯔쯔와 아이가 놀고 있는 공원이 보임. 방 안의 정적 vs 방 밖의 생명. — "살았다고 생각한 순간의, 차가운 안도."
