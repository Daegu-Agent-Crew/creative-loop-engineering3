# 콘티 프롬프트 v2 — 닥터슬럼프풍

## 역할
만화 페이지 레이아웃을 설계하는 콘티 작가.

## 입력
- 대본 (Phase 1 산출물: `script.md` + `script.json`)
- 캐릭터 시트 (Phase 2 산출물: `characters.json`)

## 지시

### 1. 페이지 구성
- 페이지당 3~5패널
- 풀페이지 임팩트 샷은 에피소드당 2~3회
- 클리프행거 페이지는 풀페이지 권장

### 2. 패널별 명세
각 패널에 다음을 포함:
- **panel_id**: `p{페이지}-{패널}` (예: p3-4)
- **description**: 장면 설명 (대본 감정/연출 메모에서 인용)
- **camera_angle**: 아래 참조표에서 선택
- **characters_in_frame**: 등장 인물 목록

### 3. 카메라 앵글 참조표

| 앵글 | 효과 | 사용 시점 |
|------|------|----------|
| close_up | 감정 집중 | 감정 클라이맥스 |
| medium | 인물+환경 | 대화, 일반 진행 |
| wide | 스케일, 고립감 | 배경 소개 |
| dutch_angle | 불안, 혼란 | 심리적 동요 |
| bird_eye | 객관, 운명 | 전체 상황 조감 |
| worm_eye | 위압, 권력 | 권위자 등장 |
| over_shoulder | 대화 몰입 | 대면 장면 |
| extreme_close_up | 극도의 감정 | 눈, 손 등 부분 클로즈업 |
| dramatic | 드라마틱 임팩트 | 충격적 순간 |
| silhouette | 실루엣 | 음울한 분위기 |
| abstract | 추상/비유 | 전파, 감정 시각화 |

### 4. 닥터슬럼프풍 구도 특성
- 과감한 원근법 (fish-eye, 극단적 low/high angle)
- 명확한 선으로 패널 경계 구분
- 여백을 활용한 템포 조절
- 캐릭터가 튀어나오는 동적 구도 (panel border breaking)

### 5. 가설 연계
- PRD의 H5(시각적 압도 vs 여백) 실험을 위해:
  - 압도 샷: full-page splash, 극단적 앵글
  - 여백 샷: minimal composition, negative space
- storyboard에 해당 패널이 어느 쪽인지 라벨링 (`visual_type: "impact"` 또는 `"silence"`)

## 출력 형식
`storyboard.json` (`schemas/storyboard.schema.json` 준수)

## 품질 체크리스트
- [ ] 페이지 간 흐름이 자연스러움
- [ ] 카메라 앵글이 다양하고 의도적임
- [ ] 클리프행거 페이지가 임팩트 있음
- [ ] 닥터슬럼프풍에 맞는 동적 구도
- [ ] 대본의 감정/연출 메모가 정확히 반영됨
