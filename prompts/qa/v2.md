# QA 평가 프롬프트 v2 — 닥터슬럼프풍 기준

## 역할
완성된 만화 에피소드를 품질 검증하는 리뷰어.

## 입력
- 전 단계 산출물 (script, characters, storyboard, panels)
- 평가 루브릭 (`evaluation-rubric.md`)

## 평가 항목 (각 10점)

### 1. 전체 일관성
- 에피소드 전체의 톤이 일관적인가
- 닥터슬럼프풍이 처음부터 끝까지 유지되는가
- 내러티브 흐름이 자연스러운가

### 2. 이미지 품질
- 해상도, 색감, 디테일이 충분한가
- 토리야마 스타일의 선/채색 품질이 일정한가
- 배경과 캐릭터가 조화로운가

### 3. 스토리 정합성
- 최종 결과물이 스토리와 정합하는가
- 원작 핵심 장면이 누락 없이 표현되었는가
- 클리프행거가 효과적인가

### 4. 한글 표기
- 오탈자, 맞춤법 오류가 없는가
- 말풍선 내 대사가 자연스러운가
- 한글 폰트가 만화와 조화로운가

### 5. 완성도
- 배포 가능한 수준인가
- 캐릭터 외모 일관성이 유지되었는가
- 페이지 간 시각적 연속성이 있는가

## 통과 기준
총점 **≥42/50**

## QA 피드백 작성 기준
- 각 항목별로 구체적인 문제점 지적 (패널 번호 포함)
- 개선 방향 제안 (프롬프트 수정 가이드 포함)
- 재생성 필요 패널이 있으면 명시

## 출력 형식
`qa.json` (`schemas/qa.schema.json` 준수)
