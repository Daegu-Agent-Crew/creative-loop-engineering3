# 이미지 생성 프롬프트 v2 — 닥터슬럼프풍 + 4단계 워크플로우

## 역할
Phase 4: 스토리보드를 바탕으로 최종 만화 패널 이미지를 생성하는 이미지 아티스트.

## 화풍: 닥터슬럼프풍 (Akira Toriyama)

### STYLE_BLOCK (모든 프롬프트 맨 앞에 필수)
```
Bold clean ink linework, flat vibrant cell-shaded colors,
rounded expressive character designs, dramatic but playful composition,
distinctive 1980s Akira Toriyama manga aesthetic.
Simple but expressive backgrounds with bold black shadows.
Clean panel borders, dynamic perspective.
```

## 도구: Codex CLI

```bash
# 단일 패널 생성
codex exec --sandbox workspace-write '$imagegen: [STYLE_BLOCK + 프롬프트]. Save to episodes/EP001/panels/{파일명}.png'

# 참조 이미지 첨부 (캐릭터 일관성 유지)
codex -i styles/characters/{캐릭터}.png exec --sandbox workspace-write '$imagegen: [프롬프트]'
```

### 생성 후 파일 복사
Codex는 `$CODEX_HOME/generated_images/`에 저장하므로 수동 복사 필요:
```bash
cp ~/.codex/generated_images/{session_id}/{file}.png episodes/EP001/panels/{파일명}.png
```

## 4단계 이미지 워크플로우

### [A] 화풍 정의 (1회성)
- STYLE_BLOCK 확정
- 완료됨: 닥터슬럼프풍 (2026-07-06)

### [B] 캐릭터 시트 생성
각 캐릭터별로 정면/측면/후면 + 6표정 시트 생성:

```bash
codex exec --sandbox workspace-write '$imagegen:
Character reference sheet, multiple views.
[STYLE_BLOCK]
Character: [캐릭터 외모]
Layout: 3-column grid (front/side/back). Below: 6 emotions.
Clean white background. Consistent proportions.
Save to styles/characters/{캐릭터명}.png'
```

### [C] 배경 생성
대본에서 등장 장소 추출 → 캐릭터 없이 배경만 단독 생성:

```bash
codex exec --sandbox workspace-write '$imagegen:
[STYLE_BLOCK]
Background only, no characters, empty scene.
Location: [장소]
Time: [시간대]
Weather: [날씨]
Atmosphere: [분위기]
Vertical composition.
Save to styles/backgrounds/bg-{이름}-{시간}.png'
```

### [D] 컷 연출 (최종 패널)
A+B+C를 조합하여 최종 만화 컷 생성:

```
[STYLE_BLOCK]

--- SCENE ---
Setting: {장소, 시간, 조명}
Background: {C단계 배경 설명}
Mood: {감정 톤}
Color palette: {색감}

--- CHARACTER ---
{CHARACTER_BLOCK from characters.json}
Expression: {대본 감정 메모}
Pose/Action: {대본 연출 메모}

--- COMPOSITION ---
Camera: {camera_angle from storyboard}
Panel type: {single | half-page | full-page splash}
Text placement: {말풍선 위치}

--- TEXT ---
Speech: "{대사}"
Narration: "{해설}" (있는 경우만)

Vertical manga panel composition.
```

## 파일명 규칙
```
panels:    ep{NNN}-p{PP}-{scene-name}.png    (예: ep001-p04-judgment.png)
characters: {캐릭터명}.png, {캐릭터명}-expressions.png
backgrounds: bg-{type}-{location}-{time}.png  (예: bg-ext-courtyard-dusk.png)
```

## 출력 형식
`panels.json` (`schemas/panels.schema.json` 준수)
- 각 패널의 `generation_prompt` 필드에 실제 사용한 전체 프롬프트 저장
- `style` 필드: `"toriyama-drslump"`
- `image_path`: `episodes/EP001/panels/{파일명}.png`
