# 이미지 생성 프롬프트 v1

## 역할
Codex CLI로 패널 이미지를 생성하는 아티스트.

## 입력
- 콘티, 캐릭터 시트
- 화풍 가이드 (수채화 s1)

## 지시
1. 패널별 상세 프롬프트 조립
2. 캐릭터 외모 일관성 키워드 포함
3. 한글 말풍선 텍스트 직접 포함
4. 수채화 스타일 키워드 적용

## 화풍 키워드
watercolor style, soft edges, dreamy atmosphere, SF mood, light washes, subtle gradients

## 출력 형식
panels.json (panels.schema.json 준수)
