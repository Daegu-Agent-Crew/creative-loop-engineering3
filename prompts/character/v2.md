# 캐릭터 디자인 프롬프트 v2 — 닥터슬럼프풍

## 역할
삼체 만화의 캐릭터를 닥터슬럼프풍(Akira Toriyama style)으로 디자인하는 캐릭터 디자이너.

## 화풍 고정 블록
모든 캐릭터 설명에 아래 스타일을 적용:

```
Bold clean ink linework, flat vibrant cell-shaded colors,
rounded expressive character designs, dramatic but playful composition,
distinctive 1980s Akira Toriyama manga aesthetic.
Simple but expressive backgrounds with bold black shadows.
```

## 입력
- 스토리 대본 (Phase 1 산출물)
- 기존 캐릭터 위키 (`wiki/characters/`)

## 지시

### 1. 캐릭터별 외모 명세
- 나이, 체형, 머리스타일, 눈 모양, 의상, 액세서리
- 닥터슬럼프풍 특성: 둥근 윤곽, 과장된 표정, 굵은 선
- 캐릭터간 시각적 구분성 확보 (실루엣만 봐도 구분 가능)

### 2. 토리야마 스타일 변환 포인트
- 눈: 크고 둥글거나 날카로운 삼각형 — 캐릭터별 차별화
- 머리: 단순화된 형태, 인위적이지만 캐릭터적인 헤어스타일
- 체형: 약간 과장된 비율 (큰 머리, 작은 체형 또는 묵직한 체형)
- 의상: 단순한 색 블록, 큰 단추나 문장 등 시그니처 요소
- 표정: 과장된 감정 표현 — 화내면 불꽃, 울면 분수처럼 눈물

### 3. 색감 지정
각 캐릭터별 코어 색상 2-3종 지정:
- 예원제: deep blue, white, silver accents
- 스치친: olive brown, dirty white, amber
- 청신: soft lavender, grey, warm white
- 왕먀오: steel blue, cool grey, lens reflection
- 장베이하이: navy, gold, black

### 4. style_notes 작성 기준
`characters.json`의 `style_notes` 필드는 Phase 4 이미지 생성에 직접 사용됨:
```
"style_notes": "Toriyama style: large round eyes, simplified hair spikes, cell-shaded uniform with bold button details. Cheeky but determined expression."
```

## 캐릭터 시트 생성 (Phase 4 [B] 단계)

### Codex CLI 실행
```bash
codex exec --sandbox workspace-write '$imagegen:
Character reference sheet, multiple views.
Bold clean ink linework, flat vibrant cell-shaded colors,
rounded expressive character designs, 1980s Akira Toriyama manga aesthetic.

Character: [캐릭터 외모 명세]
Layout: 3-column grid. Left: front view. Center: side profile. Right: back view.
Below: 6 emotion variants (neutral, angry, sad, surprised, determined, fearful).
Full body, clean white background, no environment.
Consistent proportions across all views.
Save to styles/characters/{캐릭터명}.png'
```

## 출력 형식
`characters.json` (`schemas/characters.schema.json` 준수)

## 품질 체크리스트
- [ ] 각 캐릭터가 닥터슬럼프풍으로 통일
- [ ] 캐릭터 간 시각적 구분성
- [ ] style_notes가 Phase 4에서 직접 사용 가능한 수준
- [ ] 원작 캐릭터 성격이 외모에 반영됨
