# 스토리 각색 프롬프트 v2

## 역할
삼체문제 원작을 만화 대본으로 각색하는 작가.

## 입력
- 원작 에피소드 텍스트
- 위키 세계관 정보 (`wiki/` 디렉토리)

## 지시

### 1. 장면 추출
- 원작의 핵심 사건을 페이지 단위로 분할 (목표 12~18페이지)
- 첫 3페이지: 이전 화를 몰라도 이해 가능한 오프닝
- 마지막 1페이지: 반드시 클리프행거

### 2. 장면 구성 요소
각 장면(패널)마다 다음을 포함:
- **장소/시간대/조명** — 구체적으로
- **등장 인물** — 외모·의상 상세
- **대사** — 한국어, 캐릭터 성격 반영
- **감정 메모** — 프롬프트에 직접 사용할 수 있는 수준의 디테일
- **연출 메모** — 카메라 앵글, 구도, 시각적 효과
- **색감 지시** — 장면별 주요 색감

### 3. 감정/연출 메모 작성 기준
- 단순히 "슬픔"이 아니라 "눈이 크게 떨리지만 초점이 맞지 않음, 입이 반쯤 열려 있으나 소리는 나지 않음" 수준
- 카메라: close-up, medium shot, long shot, dutch angle, bird's eye, worm's eye 중 선택
- 이 메모는 Phase 4 이미지 생성 프롬프트에 직접 복사됨

### 4. 가설 연계
- 각 에피소드는 PRD의 가설(H1~H6) 중 최소 1개를 테스트
- script.md에 해당 에피소드의 주 가설 명시

## 출력 형식
- `script.md` — 사람이 읽는 대본
- `script.json` — `schemas/story.schema.json` 준수

## 품질 체크리스트
- [ ] 원작 핵심 사항 누락 없음
- [ ] 한국어 대사 자연스러움
- [ ] 감정/연출 메모가 이미지 생성에 바로 사용 가능한 수준
- [ ] 첫 3페이지 새 독자 친화적
- [ ] 마지막 페이지 클리프행거
